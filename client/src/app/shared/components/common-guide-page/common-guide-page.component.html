Добавление

Добавление нового элемента списка (слова или фразы) - кнопка Add new . Стоит обратить внимание, что при добавлении некоторых элементов (неправильные глаголы, личный словарь), поле транскрипция является обязательным для заполнения. Если же транскрипция не имеет значения - то в поле ввода можно поместить пробел, "-" или любой другой символ.

Также следует обратить внимание: при добавлении слова в личный словарь (раздел Private vocabulary) удобно воспользоваться ссылкой Convenient resource. Для этого нужно ввести слово (поле new word), а затем перейти по ссылке. После этого, в открывшейся новой вкладке, можно скопировать американский или английский вариант транскрипции данного слова и его перевод и поместить их в соответствующие поля личного словаря.

Для того, чтобы вернуться к исходному списку, можно воспользоваться кнопкой Return , кнопкой браузера "назад", либо перейти к другому пункту меню.

Редактирование и удаление

Для редактирования или удаления элемента списка необходимо кликнуть по соответствующей строке таблицы и, после перехода на страницу редактирования, либо отредактировать данные и нажать на кнопку Save , либо, для удаления слова, нажать на кнопку Delete .

Сортировка списка

Сортировать список можно либо по алфавиту (order by alphabet) либо по дате (order by date) - в этом случае вверху списка будут последние добавленные элементы.

Поиск (фильтрация) по списку

Для фильтрации списка можно пользоваться полем search, куда нужно ввести условие фильтрации (первую букву либо начальную часть слова). Для очистки поля поиска нужно использовать кнопку с изображением совка.

Удаление списка

Удаление списка - кнопка Delete list . Есть два варианта: удалить список полностью(completely), либо только его часть(selectively).

При выборе варианта удаления Selectively , необходимо отметить галочками те слова, которые нужно удалить.

Загрузка дефолтного списка

Если список элементов пуст можно загрузить некий дефолтный список (и при необходимости отредактировать его). Для этого нужно воспользоваться кнопкой Download default list
